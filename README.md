Laika
====

Responsive Boilerplate

## Features

### HTML
* Semantic HTML5 markup
* Responsive ready
* Uses BEM syntax for class names
* Uses the Bulma CSS framework based on Flexbox

### CSS
* SASS Ready

### NPM Requirements
* bulma
* gulp
* jshint
* sass
* terser
* concat
* rename
* plumber
* cssnano
